# Support

## Where to get help

1. **Documentation**: Check the [README](README.md) and example configurations in `examples/`.
2. **Issues**: If you find a bug or have a feature request, please open an issue on [GitHub](https://github.com/darshan-rambhia/terraform-provider-filesync/issues).

## Community

If you have questions or want to discuss the provider, feel free to open a discussion on GitHub (if enabled) or file an issue with the "question" label.
